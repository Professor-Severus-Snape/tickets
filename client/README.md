# Frontend

Клиентская часть приложения «Менеджер тикетов».  
Описание проекта — в [главном README](../README.md).

## Назначение

- Отображение списка тикетов
- Создание, редактирование и удаление задач
- Просмотр деталей тикета
- Изменение статуса выполнения

## Технологии

- JavaScript (ES6+)
- DOM API
- CSS
- Webpack
- ESLint (Airbnb config)
